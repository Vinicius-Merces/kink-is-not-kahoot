# 🔥 KINK is not Kahoot

*The rebellious quiz platform that breaks all the rules*

## 🎯 Mission

KINK is not Kahoot. We're the alternative you've been waiting for. 
No limits. No restrictions. Pure competitive fun.

## ✨ Features

- **Unlimited Questions**: Create quizzes with as many questions as you want
- **No Character Limits**: Write questions and answers without restrictions
- **Real-time Gameplay**: Instant responses and live rankings
- **Speed-Based Scoring**: Points based on answer speed and accuracy
- **Custom Avatars**: Students choose their identity
- **Teacher Dashboard**: Full control over quizzes and live sessions
- **No Student Registration**: Just a nickname and you're in!

## 🚀 Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Firebase (Authentication, Firestore, Cloud Functions)
- **Real-time**: Firestore listeners
- **Hosting**: squareweb, in a near future AWS

## 🛠️ Setup

1. Clone the repository
2. Create a Firebase project
3. Enable Authentication (Email/Password + Google)
4. Enable Firestore
5. Update `firebase-config.js` with your credentials
6. Run `firebase deploy`

## 🎮 How to Play

### For Teachers (Hosts)
1. Create an account
2. Create your quiz with questions and answers
3. Start a live session
4. Share the 6-digit code with students
5. Control the game flow

### For Students (Players)
1. Enter the 6-digit code
2. Choose a nickname and avatar
3. Answer questions in real-time
4. Compete for the top spot!
